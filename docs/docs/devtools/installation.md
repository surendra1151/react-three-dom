# DevTools Installation

The Chrome DevTools extension adds an **R3F** tab for visual scene inspection.

Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/react-three-dom/knmbpbojmdgjgjijbepkmgpdklndlfkn).

## Verify

1. Open any page running your R3F app with `<ThreeDom />`
2. Open Chrome DevTools (F12)
3. Look for the **R3F** tab in the DevTools panel tabs
